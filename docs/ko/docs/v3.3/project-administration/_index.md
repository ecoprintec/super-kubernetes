Super Kubenetes 프로젝트는 쿠버네티스 네임스페이스입니다. 프로젝트에는 단일 클러스터 프로젝트와 멀티 클러스터 프로젝트의 두 가지 유형이 있습니다. 전자는 일반 쿠버네티스 네임스페이스이고 후자는 여러 클러스터에 걸친 연합 네임스페이스입니다. 프로젝트 관리자는 프로젝트 생성, 범위 제한 설정, 네트워크 격리 구성 등을 담당합니다.
